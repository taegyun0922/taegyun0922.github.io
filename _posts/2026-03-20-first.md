## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 나는 어떤 사람일까? — MBTI로 나를 알아보자 |

## 2. MBTI 테스트 페이지 구성

### 2-1. 페이지 흐름 (UX Flow)

```
[시작 화면]
    ↓
[MBTI란? 간단 소개 (1~2줄)]
    ↓
[질문 응답 (12~20문항)]
    ↓
[결과 로딩 애니메이션]
    ↓
[결과 페이지 — 나의 MBTI 유형]
    ↓
[상세 설명 / 궁합 / 직업 추천 보기]
    ↓
[공유하기 버튼]
```

### 2-2. 질문 구성 (4가지 축)

| 축 | 대립 성향 | 예시 질문 |
|----|-----------|-----------|
| E / I | 외향 / 내향 | "주말에 친구들과 약속이 잡혔다. 나는?" |
| S / N | 감각 / 직관 | "새로운 것을 배울 때 나는?" |
| T / F | 사고 / 감정 | "친구가 고민을 털어놓을 때 나는?" |
| J / P | 판단 / 인식 | "여행 계획을 짤 때 나는?" |

- 문항 수: 각 축당 3~5문항 → 총 **12~20문항** 권장
- 형식: A / B 양자택일 또는 5점 척도 슬라이더
- 진행 표시: 상단 프로그레스 바 표시

### 2-3. 결과 페이지 구성 요소

#### 🎨 디자인 컨셉: 캐릭터 + 말풍선

> 유형별 전용 캐릭터가 말풍선으로 나의 특징을 직접 말해주는 형식  
> 텍스트 나열 대신 대화하듯 읽히는 친근한 UX

```
┌─────────────────────────────────┐

│   (유형별  │  💬 말풍선 2        │
│    전용)   │  "감정에 솔직하고  │  ← 특징 2
│            │   공감 능력이 높아요"│
│            ├────────────────────┤
│            │  💬 말풍선 3        │

└────────────┴────────────────────┘
│  ❤️ 잘 맞는 유형      ⚡ 조심할 유형  │  ← 궁합 섹션
│  ENFJ, ENTJ          ESTP, ESTJ    │
├─────────────────────────────────┤
│  💼 추천 직업                        │  ← 직업 추천 섹션

├─────────────────────────────────┤
│  📤 결과 공유하기                    │  ← 공유 버튼
│  [카카오톡]  [링크 복사]  [이미지저장]│
└─────────────────────────────────┘
```

#### 🐾 캐릭터 디자인 가이드

| 항목 | 내용 |
|------|------|
| 스타일 | 동글동글한 2D 캐릭터 (동물 or 이모지 스타일 권장) |
| 유형별 차별화 | 유형마다 다른 동물 캐릭터 또는 의상/표정으로 구분 |
| 표정 | 유형 특징을 반영한 표정 (예: INFP → 몽상적인 표정, ENTJ → 당당한 표정) |
| 컬러 | 유형 그룹별 대표 색상 적용 (아래 색상 가이드 참고) |

#### 🎨 유형 그룹별 컬러 가이드

| 그룹 | 유형 | 대표 컬러 | 분위기 |
|------|------|-----------|--------|
| 분석형 (NT) | INTJ, INTP, ENTJ, ENTP | 보라 / 남색 | 차갑고 지적인 느낌 |
| 외교형 (NF) | INFJ, INFP, ENFJ, ENFP | 초록 / 민트 | 따뜻하고 감성적인 느낌 |
| 관리자형 (SJ) | ISTJ, ISFJ, ESTJ, ESFJ | 파랑 / 하늘 | 안정적이고 신뢰감 있는 느낌 |
| 탐험가형 (SP) | ISTP, ISFP, ESTP, ESFP | 주황 / 노랑 | 활기차고 자유로운 느낌 |

---

#### 💬 말풍선 콘텐츠 구성 (유형당 5개)

각 말풍선에 들어갈 내용 유형:

1. **핵심 성격** — "저는 ○○한 편이에요"
2. **일상 행동 패턴** — "○○할 때 이런 모습이 나와요"
3. **장점 어필** — "제 강점은 ○○이에요!"
4. **약점 고백** — "사실 ○○하는 건 좀 힘들어요 😅"
5. **한 마디 요약** — 유형을 대표하는 명언 or 짧은 문장

> 💡 말풍선은 캐릭터 오른쪽 또는 위에 순서대로 등장하는 애니메이션 효과 추가 권장

## 4. 콘텐츠 상세 계획

### 4-1. 1편 — MBTI 입문 소개 포스팅 구성

```
## MBTI가 뭔가요?
- 탄생 배경 (마이어스-브릭스, 융의 심리 유형론)
- 4가지 축 설명 (E/I, S/N, T/F, J/P)
- MBTI의 활용 방법

## 오해와 진실
- MBTI는 절대적이지 않다
- 사람은 변한다 — 환경에 따라 달라지는 성향

## 테스트 바로 가기 [링크]
```

### 4-2. 3편 — 16가지 유형 상세 설명 구성

각 유형별로 아래 항목을 카드 형태로 작성:

- **유형명** (예: INFP)
- **별명** (예: 열정적인 중재자)
- **핵심 키워드** 3개
- **장점** 3가지
- **단점** 3가지
- **이런 사람이에요** (상황 묘사 1~2문장)
- **대표 유명인/캐릭터**

> 💡 16개 유형을 4개 그룹(분석형 NT / 외교형 NF / 관리자형 SJ / 탐험가형 SP)으로 묶어서 소개하면 입문자가 이해하기 쉬움

### 4-3. 4편 — 궁합 & 관계 포스팅 구성

```
## 궁합이 잘 맞는 유형 조합 TOP 5
## 오해하기 쉬운 유형 조합
## 직장에서 잘 맞는 유형 vs 충돌하는 유형
## 연인 관계에서 유형별 특징
## 궁합표 (16×16 매트릭스 이미지)
```

### 4-4. 5편 — 직업 & 진로 추천 구성

| 유형 그룹 | 대표 직업군 |
|-----------|------------|
| 분석형 (NT) | 개발자, 과학자, 전략가, 기획자 |
| 외교형 (NF) | 상담사, 작가, 교사, 사회복지사 |
| 관리자형 (SJ) | 회계사, 공무원, 관리직, 의료직 |
| 탐험가형 (SP) | 예술가, 운동선수, 디자이너, 요리사 |
